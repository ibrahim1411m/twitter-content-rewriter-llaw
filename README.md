# 🏢 Twitter → محتوى عقاري تثقيفي

أداة تجلب تغريدات حسابات تويتر المختارة وتعيد صياغتها بأسلوب تثقيفي عقاري، ثم تصدّرها كملف Excel وMarkdown.

---

## 💰 التكلفة

- **Apify**: $0.25 لكل 1000 تغريدة (مع $5 رصيد مجاني شهرياً)
- **Anthropic**: ~$0.30 لإعادة صياغة 100 تغريدة (مع $5 رصيد مجاني عند التسجيل)
- **2000 تغريدة شاملة الكل**: ~$1

---

## 🚀 التشغيل من GitHub

### الخطوة ١ — إضافة المفاتيح السرية

اذهب إلى: **Settings → Secrets and variables → Actions → New repository secret**

| الاسم | المصدر |
|-------|--------|
| `ANTHROPIC_API_KEY` | [console.anthropic.com](https://console.anthropic.com) |
| `APIFY_TOKEN` | [console.apify.com](https://console.apify.com/account/integrations) |

---

### الخطوة ٢ — تشغيل العملية

1. تبويب **Actions** → **جلب وإعادة صياغة المحتوى**
2. **Run workflow** → حدد الحساب والعدد → **Run workflow**

---

### الخطوة ٣ — النتائج

```
content/
├── README.md                          ← فهرس
└── ahmed_alshuhail/
    └── 2025-06-03/
        ├── ahmed_alshuhail_2025-06-03.xlsx   ← Excel
        └── ahmed_alshuhail_2025-06-03.md     ← تقرير
```
